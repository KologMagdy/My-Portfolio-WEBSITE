# Quickstart

Because this feature comprises CSS layout adjustments, there are no new backend or build dependencies to install.

To view the changes:
1. Ensure you have the repository downloaded.
2. Open `index.html` in your preferred modern web browser.
3. Switch the language toggle between English and Arabic to verify RTL/LTR text wrapping behaviors.
4. Scale your browser width simulating down to 320px viewport width (Mobile S) to verify word breaking and line clamping.
